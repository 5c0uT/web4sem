# Security Policy

## Supported Versions

This project is a student lab repository. Only the latest state of the default
branch is considered maintained.

## Reporting a Vulnerability

If you believe you have found a security issue:

1. Do not open a public issue describing the vulnerability.
2. Contact the repository owner/maintainer via GitVerse (private message) or
   the communication channel used for the course.
3. Include steps to reproduce and any relevant logs/screenshots.
