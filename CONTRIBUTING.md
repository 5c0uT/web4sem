# Contributing

This repository is primarily used for educational labs.

## How To Participate

- If you are not the owner of this repository, please do not open PRs unless
  you have been asked to do so by the maintainer/teacher.
- If you found an issue in the code, open an Issue with:
  - a short description
  - steps to reproduce (if applicable)
  - expected vs actual behavior

## Development Notes

- Default branch `main` contains only navigation/overview (README).
- Lab solutions live in separate branches (e.g. `lab1`, `frontend`).
- For `lab1`: open `lab1/index.html` in a browser and check DevTools Console.

## Commit Messages

Use short, descriptive messages, for example:

- `Docs: update README`
- `Fix(lab1): correct output formatting`
- `Feat(lab2): add tool`
