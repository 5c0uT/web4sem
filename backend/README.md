# lab9

Итоговая версия приложения для лабораторной 9.

## Запуск

```bash
npm install
npm run serve
```

Откройте http://127.0.0.1:3000/.

## Структура

- `client` — статическая клиентская часть, которую отдает Node.js сервер.
- `server` — сервер, контроллеры, сервисы, репозиторий SQLite и SQL-схема.
