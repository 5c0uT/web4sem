# Лабораторная работа 3 - Three.js

## Цель работы

Изучить основы трехмерной графики в браузере с использованием библиотеки **Three.js**.

## Что реализовано

- одна сцена Three.js;
- четыре камеры:
  - одна перспективная для 3D-представления;
  - три ортографические для видов сверху, спереди и сбоку;
- четыре отдельных canvas для вывода каждой проекции;
- вспомогательные элементы сцены:
  - `AxesHelper`;
  - две сетки `GridHelper`;
  - горизонтальная плоскость;
- объекты сцены:
  - куб;
  - сфера;
  - треугольная пирамида, созданная вручную через `BufferGeometry` и `BufferAttribute`;
- материалы `MeshLambertMaterial`;
- источник света `HemisphereLight`;
- управление объектами через форму:
  - выбор объекта;
  - изменение цвета;
  - изменение координат `X`, `Y`, `Z`;
- перемещение выбранного объекта с клавиатуры:
  - стрелки — перемещение по плоскости `X/Z`;
  - `Page Up` / `Page Down` — перемещение по оси `Y`;
- управление камерой в окне 3D с помощью `OrbitControls`:
  - вращение;
  - масштабирование;
  - панорамирование.

## Используемые файлы

- `index.html` — структура страницы и элементы управления;
- `styles.css` — оформление страницы и панелей просмотра;
- `main.js` — создание сцены, камер, объектов и логики управления;
- `three.module.js`, `three.core.js`, `OrbitControls.js` — локальные файлы библиотеки Three.js.

## Запуск проекта

1. Откройте проект в редакторе кода.
2. Запустите локальный сервер, например через расширение **Go Live** в VS Code.
3. Откройте страницу `index.html` в браузере.

## Проверка требований

- код разбит по логическим функциям;
- HTML не генерируется строками из JavaScript;
- в консоли браузера не должно быть ошибок;
- выполнены требования лабораторной по сцене, камерам, объектам, свету и управлению.
